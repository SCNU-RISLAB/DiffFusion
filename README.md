# DiffFusion

![DiffFusion Framework](assets/fig2.pdf)

Official repository for the paper:

**Diffusion-Based Restoration for Multi-Modal 3D Object Detection in Adverse Weather**

## Overview
DiffFusion is a robust multi-modal 3D object detection framework for adverse weather conditions.  
This work introduces diffusion-based restoration and adaptive cross-modal fusion to improve robustness under challenging weather such as rain, fog, and strong sunlight.
